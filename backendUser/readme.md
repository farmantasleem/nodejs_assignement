## Api Endpoints

Register <br>
Method: POST <br>
Endpoint: /register <br>
Payload: <br>
{ "name": "user_name", "email": "user_email", "password": "user_password" } <br>
Response : <br>
{"msg":"User Registered Successfully"} <br>

LOGIN <br>
Method: POST <br>
Endpoint: /login <br>
Payload: <br>
{ "email": "user_email", "password": "user_password" } <br>
Response: <br>
{"msg":"User Login Successfully"} <br>
